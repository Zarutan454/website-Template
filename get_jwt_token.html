JWT Token Generator

Teste die Django API Login-Funktion und erhalte einen JWT Token für WebSocket-Tests.

Anleitung:

  1. Stelle sicher, dass der Django Backend auf Port 8000 läuft
  2. Gib deine Login-Daten ein
  3. Klicke auf "Login & Token generieren"
  4. Kopiere den Token und verwende ihn in der WebSocket-Test-Seite
